# Guinea-Bit-Player
App nghe nhạc native cho hệ sinh thái Apple (macOS, hướng tới iOS), tập trung vào phát bit-perfect thật qua CoreAudio HAL (Exclusive/Hog Mode) và duyệt được thư viện cực lớn (8.5TB / 11k album / 150k track) nhờ lazy browse. Kết nối media server hoặc quét nhạc trên đĩa, quản lý playlist, phát như một music center.
